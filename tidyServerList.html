TidyServerList

TidyServerList is a theme for BetterDiscord that makes the server list a lot cleaner and tidier, improved functionality in reduced space!



How to install:

  • Step 1, download the file by clicking the download button.
  • Step 2, go to your themes folder by navigating in discord to (Settings->BetterDiscord->Themes->Open Themes Folder)
  • Step 4, put the file in the themes folder
  • Step 5, go to discord and reload discord using
    Windows: CTRL+R
    Mac: CMD+R

    • How to update:

  • Step 1, remove the current theme file from your themes folder.
  • Step 2, see How to install.
